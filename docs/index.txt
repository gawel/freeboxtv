=====================================

.. include:: ../README.txt

Requirements
------------

You need vlc_

Installation
------------

Do not expect it to run on windows. I you need this, patch it yourself.

Install freeboxtv with easy_install::

  $ sudo easy_install -U freeboxtv

Troubleshooting
---------------

If you having problems, run::

  $ fbxtv -d

And add an `issue <https://github.com/gawel/freeboxtv/issues>`_ (you can use french..)
